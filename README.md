# Detailed Canvas

Enrich canvas link cards with AI-generated descriptions and cover images using Ollama.

## Features

- AI-generated descriptions for canvas link cards
- Custom cover images generated with Ollama
- Seamless integration with Obsidian Canvas

## Installation

### Manual Installation

1. Download the latest release from GitHub
2. Extract the files to your vault's `.obsidian/plugins/detailed-canvas/` directory
3. Reload Obsidian
4. Enable the plugin in Settings â†’ Community Plugins

## Development

```bash
# Install dependencies

# Lint the code
```

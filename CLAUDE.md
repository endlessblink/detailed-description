# Detailed Canvas - Obsidian Plugin

## Obsidian Vault

After building (`npm run build`), always copy the updated plugin to the vault so it can be tested:

```bash
cp main.js "/media/endlessblink/data/app-data/sync/Dropbox/OBSIDIAN_SYNCED/MAIN VULT/.obsidian/plugins/detailed-canvas/main.js"
```

The user must then reload the plugin in Obsidian (Ctrl+P → "Reload app without saving" or toggle the plugin off/on in Settings → Community Plugins).
